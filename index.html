March 2015. TransitBPM
java ee  activiti bpm  spring framework  spring security  bootstrap  jpa  mysql  atomikos  chartjs

I have developed a full business process management web tool, using Activiti as BPM.
It includes:
- Management of users, groups and rights.
- Design of business process using Activiti Modeler.
- A simple process designer using tables.
- A control panel for know how the process are running and for discovering bottlenecks: time per task, number of instances per task, finished vs running,...

July 2015 - present. +Retina
e-health  java ee  activiti bpm  spring framework  spring security  bootstrap  jpa  mysql  atomikos  chartjs

+Retina is a web application based in TransitBPM for manage optical diagnosis.
Its features are:
- Management of patients, optics, doctors and cases.
- Optometry images management.
- Doctor's report.
- Automatic mailing to the patients with the final diagnosis and doctor's report.
- A control panel for knowing how the processes are running and for discovering bottlenecks: time per task, number of instances per task, finished vs running,...
- Also, I have designed and built from the ground up the corporate website, +Retina.
